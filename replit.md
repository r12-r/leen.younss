# Leen Personal Page

## Overview
A simple, elegant personal profile page with social media links. This is a static HTML/CSS website featuring:
- Custom gradient design with pink/magenta theme
- Profile photo display
- Links to Instagram, TikTok, and WhatsApp
- Custom Designer font
- Responsive layout with hover effects

## Project Structure
- `index.html` - Main HTML page
- `style.css` - Styling with custom gradients and animations
- `Designer.otf` - Custom font file
- `icon.jpg` - Profile photo

## Technology Stack
- Pure HTML5 and CSS3
- No framework or build process required
- Static file serving

## Running the Project
The project runs a simple Python HTTP server on port 5000 to serve the static files.
